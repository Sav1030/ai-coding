# 欢迎参加 HacktoberFest 2019!

请按照以下步骤来提交你的第一个pull request:

1. Fork这个项目。

2. 把这个项目clone到你你的电脑上。

3. 从话题（issues）中选择其中一个，然后提交PR。

4. 在terminal中输入以下命令：
    ```
    git add -A
    git commit -m "your name"
    git push origin master
    ```
5. 提交pull request。

6. Star这个项目。

7. 恭喜！！你已成功创造你的PR。

8. 查看你的[进展](https://hacktoberfest.digitalocean.com/profile)等待免费T恤。
